# RECRUITER-X — Design System Index
## Electric Intelligence Aesthetic — v1.0

> **Design thesis.**
> Recruiter-X looks like the inside of a machine that has just been given a problem to solve. Electric blue on deep navy. Monospaced data. Clean. Dense. Fast. Every pixel earns its place because every second in a recruiter's day costs money.
